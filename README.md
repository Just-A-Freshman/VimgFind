# 测试分支
## 说明
这是一个新功能的测试分支。
- 由于Bug较多，且整体代码较为混乱，因此该分支代码不会打包进入release，需等到后续代码修复及测试；
- 由于期末考试时间紧张，当下不得不停工了，代码临时搁置与此；

## 新功能
- 可以选择搜索返回的结果数（指定为10/30/50/100之中的一种）
- 可以选择预览模式，新增缩略图的预览模式，大致效果如下：<img width="1034" height="674" alt="image" src="https://github.com/user-attachments/assets/eba299ac-3735-4c29-9488-0a6c4f33be52" />

## 当前BUG
- 缩略图组件的父类字体继承问题，导致初始化图标缩略图时使用了默认字体颜色而非父类字体颜色，字体和背景混为一体难以分辨；
- 从详细信息模式切换到缩略图模式时，当前选中项会被截断。例如在原始表格中多选了N个项(N > 6)，切换到缩略图模式后选中项会被截断为6个；
