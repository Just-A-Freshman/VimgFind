# 基于AI的本地版以图搜图
## 版本功能预览
- v1.0 基本的以图搜图
- v1.1 增加剪切板搜图
- v2.1 增加以文搜图(回车键触发)，但由于更新顺序原因缺少了剪切板搜图功能
- v2.2(最新) 恢复了剪切板搜图，增加了【图标预览模式】，【返回结果数控制】，右键菜单【另存为】功能
